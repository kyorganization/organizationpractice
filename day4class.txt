# ‚öñÔ∏è Conditional Statements

-> if statement  
* Allows Python to execute a block if condition is TRUE  
* If the condition is FALSE, that block is skipped  

Syntax:

    # block of code

Example:
age = 20

    print("‚úÖ You can vote")

-> if-else  
* If condition is TRUE ‚Üí if block executes  
* If condition is FALSE ‚Üí else block executes

    # block of code

Example:

    print("‚úÖ Eligible to vote")

    print("‚ùå Not eligible to vote")

-> if-elif-else ladder  
* Multiple conditions checked in sequence  

Syntax:

    # code

    # code
elif condition:
    # code
else:
    # code

Example:
marks = 65

    print("Grade A+")
elif marks >= 80:
    print("Grade A")
elif marks >= 70:
    print("Grade B")
elif marks >= 60:
    print("Grade C")
else:
    print("Fail")
