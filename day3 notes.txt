# âœ‚ï¸ Slicing
Syntax: sequence[start:end:step]

- start â†’ index where slicing begins  
- end   â†’ index where slicing stops (not included)  
- step  â†’ jump size (how many steps to skip)  

Example with string "INDIA":
I  N  D  I  A  
0  1  2  3  4  
-5 -4 -3 -2 -1  

a = "INDIA"
print(a[1:4])   # "NDI" â†’ starts at index 1, stops before 4
print(a[:4])    # "INDI" â†’ starts at 0, stops before 4
